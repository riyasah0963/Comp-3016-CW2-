ECHOES OF THE FORGOTTEN REALM - CW2 SUBMISSION
===============================================

⭐ QUICK START (Visual Studio 2022):
------------------------------------
1. Double-click: EchoesGame.sln
2. Press: F5 (or click green ▶️ button)
3. Wait: 10-30 seconds for compilation
4. Play: Game window opens automatically!

That's it! No other steps needed!

WHICH FILE TO OPEN:
-------------------
✅ OPEN THIS: EchoesGame.sln (Solution file)
❌ DON'T OPEN: Individual .cpp files, .vcxproj, or Makefile

BUILD CONFIGURATION:
--------------------
- Debug mode: Good for development
- Release mode: Best performance (RECOMMENDED for professor testing)
- Platform: x64 (should be selected by default)

FEATURES:
---------
- Full 3D OpenGL graphics
- Real-time gameplay (WASD + Mouse)
- Audio system with 3D spatial sound
- Particle effects (blood, dust, sparkles)
- 17 rooms across 7 unique biomes
- 15+ enemy types
- Enhanced combat system
- 30-60 minutes of gameplay

CONTROLS:
---------
WASD        - Move
Mouse       - Look around
E           - Interact
Left Click  - Attack
TAB         - Inventory
M           - Memories
ESC         - Quit

DEPENDENCIES:
-------------
- OpenGL 3.3+
- GLEW (static)
- GLFW3
- OpenAL
- GLM

Windows: All libraries configured in Visual Studio project
Linux: Install via: sudo apt-get install libgl1-mesa-dev libglew-dev libglfw3-dev libopenal-dev
